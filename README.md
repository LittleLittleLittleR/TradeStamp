# TradeStamp

## Dev Setup
### Backend
1. On the project folder, run:
```cmd
cd backend
uvicorn main:app
```
2. Go to http://127.0.0.1:8000/docs to test the endpoints

### Frontend
1. On the project folder, run:
```cmd
cd frontend
npm i
npm run dev
```
2. Go to http://127.0.0.1:3000 to view frontend UI
